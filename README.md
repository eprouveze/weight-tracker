## Usage

1. Clone this repo
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
4. Deploy to Vercel
5. Enable Blob storage in Vercel dashboard

## Features

- Log daily weight, waist, fasting window, energy, notes
- Simple history view
- Vercel Blob-based file storage (no DB required)
